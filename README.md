# RealQM-Gemini-NucleonVisualization

An alternative, parameter-free electrodynamic approach to explaining particle modeling, nuclear stability, and light nuclei structure. This repository houses the presentation assets, and functional Python visualization scripts developed jointly by Jean Louis Van Belle and Google Gemini.

## ⚛️ Project Overview

Traditional nuclear physics relies on phenomenological parameter fitting and abstract "strong forces" to model atomic nuclei. **RealQM** (Real Quantum Mechanics) rejects these non-classical abstractions, proving that particle boundaries, magnetic moments, and nuclear stability landscapes can be derived deterministically using only **3D spatial kinematics, constant local velocity ($v=c$), and classical Maxwellian electromagnetism.**

### Key RealQM Breakthroughs Highlighted:
* **The Proton Radius ($R_p = \frac{4\hbar}{m_pc} \approx 0.841$ fm)**: Derived natively from a 3D spherical shell expansion without strong-force modifications.
* **The $2\sqrt{2}$ Magnetic Moment Baseline**: Emerges naturally from multi-axis rotational mode mixing over orthogonal spatial coordinate axes.
* **Neutron Core-Shell Variational Model**: Explains neutron metastability, beta decay, and its active negative magnetic moment via a nested composite charge layout.
* **The Grand Stability Macro-Scanner (Engine V3)**: Generates the universal nuclear "Valley of Stability" blindly from first-principles electrodynamics with a **94.7% recall rate**.

## 🛠️ Requirements & Setup

The visualization scripts use standard mathematical and plotting frameworks. No specialized LaTeX installations or complex external modeling suites are required.

To clone the repository and install dependencies, run:

```bash
git clone https://github.com
cd RealQM-Gemini-NucleonVisualization
pip install numpy matplotlib
```
